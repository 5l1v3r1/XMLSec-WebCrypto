Um die Testbench zu Benutzen müssen folgende Schritte durchgeführt werden:
1. Starten des Webservers mit der Start_Webserver.bat
2. Aufrufen von http://localhost:8080/testbench/testbench.html

Der Benötigte nodeJs liegt fertig konfiguriert im Runtime Verzeichnis und wird mit dem Script direkt aufgerufen. 
Es ist KEINE Installation notwendig!
